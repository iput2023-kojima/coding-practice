# 2026_SolutionDeployment

## 概要

講義の振り返り内容を対話形式で入力し、レポート形式の文章を自動生成するPythonツール

## 使い方

以下のコマンドを実行してください。

```bash
python report_template.py
```

## 動作内容

プログラムを実行すると、以下のような質問が順番に表示されます。

* 講義名
* 日付
* 学んだこと
* 難しかったこと
* 次回に向けて

回答を入力すると、講義の振り返りレポートが自動生成されます。

## 実行例

```
講義名を入力してください: ソフトウェア開発演習
日付を入力してください: 2026/04/08
学んだこと: Gitの基本操作
難しかったこと: コンフリクトの解決
次回に向けて: 事前にブランチを分けて作業する

--- 生成された振り返りレポート ---

本日の「ソフトウェア開発演習」では、Gitの基本操作について学習した。
特にコンフリクトの解決が難しかった。
次回は事前にブランチを分けて作業することで改善したい。
```
