# 仕様書（spec.md）

## システム名

振り返りレポート生成ツール

## 概要

講義や作業の振り返り内容を対話形式で入力し、レポート形式の文章を自動生成するPythonツールである。
ユーザーがターミナル上で質問に回答すると、入力内容をもとに振り返りレポートを作成し画面に表示する。

## 動作環境

* Python 3.x
* ターミナル（Linux / Windows / macOS）

## 実行方法

以下のコマンドを実行する。

```bash
python3 report_template.py
```

## 入力項目

プログラム実行後、以下の項目を対話形式で入力する。

1. 名前
2. 今回の目標
3. やること
4. 出来たこと
5. 出来なかったこと
6. その理由
7. 達成度

## 処理内容

1. ユーザーに入力項目を順番に表示する
2. 入力された内容を変数に格納する
3. 入力内容をもとにレポート文章を生成する
4. 生成したレポートを画面に表示する

## 出力形式

以下のフォーマットで振り返りレポートを出力する。

```
=== 振り返りレポート ===

名前        : ○○
今回の目標  : ○○
やること    : ○○
出来たこと  : ○○
出来なかった: ○○
その理由    : ○○
達成度      : ○○

======================
```

## 関数仕様

### create_report()

振り返りレポートを生成する関数

#### 引数

なし

#### 戻り値

生成されたレポート文字列（string）

#### 処理

* 入力を受け取る
* レポート文字列を作成
* 画面表示
* 文字列を返却
